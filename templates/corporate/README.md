# Plantillas corporativas

Estas plantillas convierten los documentos de Snocomm Sentinel en entregables reutilizables para equipos de TI, dirección y área legal. Están diseñadas para uso defensivo, autorizado y proporcional.

## Plantillas incluidas

- `IT_ASSESSMENT_TEMPLATE.md`: reporte técnico para equipos TI y seguridad.
- `LEGAL_CONSENT_TEMPLATE.md`: registro de autorización, alcance y consentimiento.
- `EXECUTIVE_REPORT_TEMPLATE.md`: informe ejecutivo para dirección o gerencia.
- `RISK_MATRIX_TEMPLATE.md`: matriz de riesgo para priorización.
- `THIRTY_DAY_PLAN_TEMPLATE.md`: plan de 30 días posterior a evaluación.

## Uso recomendado

1. Ejecute Snocomm Sentinel en el equipo autorizado.
2. Adjunte o referencie `sentinel-output/results.json` y `sentinel-output/report.md`.
3. Complete la plantilla correspondiente con mínimos datos sensibles.
4. Separe hallazgos técnicos de conclusiones forenses.
5. Revise el documento con TI, legal y la persona responsable del dispositivo cuando corresponda.

## Criterio editorial

Use lenguaje sobrio: “compatible con”, “requiere revisión”, “hallazgo de riesgo”, “no concluyente”. Evite promesas absolutas, atribuciones no verificadas y exposición innecesaria de datos personales.
