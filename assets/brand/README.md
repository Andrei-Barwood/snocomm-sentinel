# Marca visual de Snocomm Sentinel

Esta carpeta contiene assets sobrios para presentar el proyecto en GitHub, documentación corporativa y materiales de comunidad.

## Archivos

- `snocomm-sentinel-mark.svg`: marca cuadrada para README, avatar secundario o documentación.
- `github-social-preview.svg`: pieza horizontal para usar como social preview o encabezado visual del repositorio.
- `readme-preview.png`: versión rasterizada y de tamaño controlado para evitar diferencias de renderizado SVG dentro del README de GitHub.

## Criterio visual

La marca usa una composición geométrica simple: escudo, triángulos alineados y un punto central de observación. La intención es comunicar defensa, claridad, evidencia y proporcionalidad, sin estética agresiva ni promesas absolutas.

## Uso recomendado

- Mantener contraste alto.
- No combinar la marca con promesas de neutralización total, detección absoluta o protección garantizada.
- Usar el encabezado en GitHub, presentaciones ejecutivas y reportes públicos.
- Mantener la marca separada de evidencia sensible o capturas de incidentes reales.
